CityScopeJS

CityScopeJS is a part of the MIT CityScope project, an open-source urban modeling and simulation tool. cityscopeJS is meant to be a unified front-end for CityScope instanecs around the world. Start by adding CityScope table name to the URL (in the form of: '...URL/?yourTableName') , for example:
CityScopeJS Hamburg, or click here for a list of active CityScope tables.
